系统维护

服务的启动和关闭

  • 关闭Nginx
$ sudo systemctl stop nginx
  • 启动Nginx
$ sudo nginx -t & sudo systemctl start nginx
  • 关闭tomcat
业务服务： 
$ /app/sites/default/corectl.sh stop
支付服务：
$ /app/sites/default/payctl.sh stop
  • 启动tomcat
业务服务： 
$ /app/sites/default/corectl.sh start
支付服务：
$ /app/sites/default/payctl.sh start

代码发布与更新

  • 用app用户登录
$ id
uid=1001(app) gid=1001(webapps) groups=1001(webapps),10(wheel)
  • 确保安装了zip和unzip
$ which zip unzip
/usr/bin/zip
/usr/bin/unzip
  • 将安装包移动至/app/deploy/package下。可能包括的几个文件：
    • ROOT.war: 核心业务服务
    • web.tar.gz: 前端页面
    • pay.war: 支付回调服务
  • 核对本地的配置文件：
DB_URL: 数据库访问地址
DB_PORT：数据库访问端口
DB_USER：数据库用户名
DB_PASS：数据库连接密码
DB_NAME：数据库名称

# Domain base
WEB_DOMAIN：对外的url名称

SITE_BASE：部署的根目录

若无特殊情况，其他设置保持不变

  • 如果有sql脚本需要执行，则先连接数据库执行sql
